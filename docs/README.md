# Documentation

Documentation policy for this repository (and, by extension, for the other
repositories of the Nova-Ideo modernisation effort):

- **Code, comments, commit messages and repository files** are written in
  **English** by default.
- [`en/`](en/) holds the **primary documentation, in English**.
- [`fr/`](fr/) holds the **French documentation**.
- **Audits, design documents and the worklog** are maintained **in both
  languages**, under the same relative path and filename in `en/` and `fr/`
  (e.g. `en/worklog.md` ↔ `fr/worklog.md`), so that each document can link to
  its counterpart.
- Whenever practical, translations are updated in the same change set as the
  original.
